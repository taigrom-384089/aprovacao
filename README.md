<h1>aprovacao</h1>
<p>Para fazer uso do projeto observer alguns itens abaixo:</p>

<p>- rodar o script database.sql em um banco de dados Sql Server.</p>

<p>- Framework utilizado: 4.5</p>
<p>- Visual Studio 2013</p>
<p>- SQL Server</p>
